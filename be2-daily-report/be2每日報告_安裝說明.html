be2 每日報告 — 安裝說明

安裝完成後，每天 09:30 自動產生你帳號下的預存金補款報告

需要的檔案（向組長索取）

🐍
be2_daily_report.py
主程式（不需要修改）
⚙️
安裝be2每日報告.command
一鍵安裝程式，雙擊執行
安裝步驟
1

將兩個檔案放在同一個資料夾

例如放在「下載項目」（Downloads）或桌面，兩個檔案必須在同一個位置。

2

右鍵點 → 「打開」安裝檔

對 安裝be2每日報告.command 按右鍵，選「打開」。
出現安全性警告時，再按一次「打開」即可。

⚠️ 若出現「無法打開」，請見下方疑難排解。

3

在終端機輸入你的帳號資訊

終端機會依序詢問：

　▸ Email：你的 kkday.com 信箱
　▸ 密碼：be2 登入密碼（輸入時不會顯示，正常現象）
　▸ 中文姓名：顯示在報告上，可留空

4

等待安裝完成

首次安裝會下載瀏覽器套件（約 3-5 分鐘），請耐心等候。
出現「安裝完成！」後，輸入 y 立即試跑一次。

✓

完成！

桌面會出現兩個捷徑：

　▸ be2每日報告.command：手動立即產生報告
　▸ 開啟最新報告.command：開啟最近一份報告

⚠️ 常見問題

「無法打開」/ 無法驗證開發者

前往 系統設定 → 隱私權與安全性，找到被擋的檔案，按「仍要打開」。

permission denied（沒有執行權限）

開啟終端機（Terminal），貼上以下指令：

bash ~/Downloads/安裝be2每日報告.command

Playwright 安裝失敗（externally-managed-environment）

開啟終端機，先跑這行再重新執行安裝：

pip install playwright --break-system-packages && python3 -m playwright install chromium
📊

安裝成功後

每天 早上 09:30 系統會自動登入你的 be2 帳號，抓取你負責的預存金帳戶資料，產生 HTML 報告並自動開啟。

報告存放位置：桌面 → be2 報告 資料夾
